  • Apps written in Objective-C
  • Run natively
  • SDK allows you to make apps

Structure of an App
Objective-C:
  1. Used by Apple and pushed by Steve Jobs
  2. A superset of C (thin layer on top)
  3. Used in Mac OS X, iPhone and iPad development
  4. Not used much elsewhere
  5. Made popular because of iOS
  6. Slightly harder than Java
Structure of an App
All code in Obj-c
Uses Interface(.m) and Header(.m)  files (like C)
Structure of an App
Frontend designed in XCode,
You never really see that code
Uses .nib files (kinda like XML)
